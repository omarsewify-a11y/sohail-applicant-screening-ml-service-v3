## Evaluation Journey

The evaluation process started with validating the initial machine learning model using standard evaluation methods:

- Accuracy
- Confusion Matrix
- Classification Report


These metrics provided evidence of the model's ability to classify applicants correctly.


## Transition to Production Monitoring

As the service evolved from a prototype into a production system, monitoring was introduced.

The monitoring layer records:

- Prediction results
- Confidence scores
- Prediction history
- System behaviour


## Final Quality Evidence

The final quality process included:

- Model evaluation
- Prediction monitoring
- Confidence analysis
- Automated alerts
